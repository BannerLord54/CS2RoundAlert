## 60 秒完整教程

标题：

CS2 新回合提醒工具，下载就能用

口播：

这个工具解决一个很小的问题：

你切出 CS2 后，新回合开始时容易忘记切回来。

使用方法很简单。

第一步，打开 GitHub Release 页面。

第二步，下载 `CS2RoundAlert.exe`。

注意，不要下载 `Source code`。

第三步，运行 exe。

第四步，点一下 `测试提示音`，确认电脑能出声。

第五步，启动或重启 CS2。

之后让这个窗口保持运行就行。

默认情况下，只有你切出 CS2 时，新回合开始才会响。

如果你想回合结束也响，勾选 `在游戏中回合结束也响`。

如果你想一整把结束后提醒休息，勾选 `整局结束后提醒休息`。

如果休息提醒不弹，点 `打开通知设置`，确认系统允许通知。

安全方面，它只使用 Valve 官方 GSI 功能。

它不会读取游戏内存，不会注入代码，不会模拟输入，也不会自动操作游戏。

这是绿色软件，不用安装。

项目开源，GitHub 地址在简介。

## 视频简介

CS2RoundAlert 是一个 Windows 小工具。

你切出 CS2 后，新回合开始会响一声，提醒你切回游戏。

下载：
https://github.com/BannerLord54/CS2RoundAlert/releases/latest

使用方法：

1. 下载 `CS2RoundAlert.exe`
2. 运行
3. 点 `测试提示音`
4. 启动或重启 CS2
5. 保持软件运行

不要下载 `Source code`，普通用户只需要下载 exe。

安全说明：
只使用 Valve 官方 GSI，不读取游戏内存，不注入，不模拟鼠标键盘，不自动操作游戏。

如果有问题或建议，请到 GitHub 提 issue。
