## 简介

CS2RoundAlert 是一个 Windows 小工具。

你切出 CS2 后，新回合开始会响一声。

下载 `CS2RoundAlert.exe`，运行，重启 CS2，就能用。

GitHub 下载：
https://github.com/BannerLord54/CS2RoundAlert/releases/latest

安全说明：
这个工具只使用 Valve 官方 GSI 功能，不读取游戏内存，不注入，不模拟鼠标键盘。

## 置顶评论

下载方法：

1. 打开 GitHub Release 页面
2. 下载 `CS2RoundAlert.exe`
3. 不要下载 `Source code`
4. 运行 exe
5. 重启 CS2

如果没声音，先点软件里的 `测试提示音`。

## 60 秒口播

CS2 死了以后，很多人会切出去看网页或者回消息。

问题是新回合开始时，经常忘了切回来。

这个工具就是解决这个问题的。

下载 `CS2RoundAlert.exe`，直接运行。

第一次打开后，它会自动查找 CS2，并写入官方 GSI 配置。

然后重启一次 CS2。

以后你切出游戏，新回合开始时，它会响一声。

如果你想让回合结束也响，可以勾选这个选项。

如果你想一整把结束后休息一下，也可以打开休息提醒。

它是绿色软件，不用安装。

它不读内存，不注入，不模拟输入，只接收 CS2 官方 GSI 数据。

项目开源，下载地址在简介。

## 视频画面顺序

1. 展示问题：切出 CS2，容易错过新回合。
2. 展示 GitHub Release 页面。
3. 点 `CS2RoundAlert.exe`。
4. 展示软件窗口。
5. 点 `测试提示音`。
6. 重启 CS2。
7. 展示切出游戏后新回合响。
8. 最后展示 GitHub 地址。
