# CS2RoundAlert v0.5.2 Release Notes

## English

CS2RoundAlert is a portable Windows tool for Counter-Strike 2.

Download `CS2RoundAlert.exe`, run it, click `Test sound`, then start or restart CS2.

Changes in this version:

- Added an English screenshot to the README.
- Shortened the notification settings button to `Open notifications`.
- Shortened the language auto option to `Auto`.
- Kept the app portable: no installer, no runtime setup.

Safety:

This tool only uses Valve's official Game State Integration. It does not read game memory, inject code, simulate input, or control gameplay.

Download `CS2RoundAlert.exe`, not `Source code`.

## 中文

CS2RoundAlert 是一个给 CS2 用的 Windows 绿色小工具。

下载 `CS2RoundAlert.exe`，运行，点 `测试提示音`，然后启动或重启 CS2。

这个版本的变化：

- README 增加英文界面截图。
- 通知设置按钮改短为 `Open notifications` / `打开通知设置`。
- 语言自动选项改短为 `Auto` / `自动`。
- 继续保持绿色软件：不用安装，不用装运行库。

安全说明：

这个工具只使用 Valve 官方 GSI。它不读取游戏内存，不注入，不模拟输入，也不会自动操作游戏。

普通用户下载 `CS2RoundAlert.exe`，不要下载 `Source code`。
