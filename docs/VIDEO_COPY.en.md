# CS2RoundAlert Video Copy

## Title

CS2 round alert when you tab out

## Description

CS2RoundAlert is a small Windows tool.

When you tab out of CS2, it plays a sound when the next round starts.

Download `CS2RoundAlert.exe`, run it, restart CS2, and keep it open.

Download:
https://github.com/BannerLord54/CS2RoundAlert/releases/latest

Safety:
It uses Valve's official Game State Integration. It does not read game memory, inject code, simulate input, or control gameplay.
