# CS2RoundAlert English Video Copy

## 30 Second Short

Title:

CS2 round alert when you tab out

Voiceover:

Do you ever tab out after dying in CS2, then miss the start of the next round?

This small tool is called CS2RoundAlert.

Download `CS2RoundAlert.exe` and run it.

On first launch, it writes the official CS2 GSI config automatically.

Restart CS2 once.

After that, when you tab out, it plays a sound when the next round starts.

It does not read game memory, inject code, simulate input, or control gameplay.

It is open source. The download link is in the description.

## 60 Second Tutorial

Title:

CS2 new round alert tool, download and run

Voiceover:

This tool solves one small problem:

You tab out of CS2, and you forget to switch back when the next round starts.

Here is how to use it.

First, open the GitHub Release page.

Second, download `CS2RoundAlert.exe`.

Do not download `Source code`.

Third, run the exe.

Fourth, click `Test sound` to make sure audio works.

Fifth, start or restart CS2.

Keep CS2RoundAlert running while you play.

By default, it only plays the new-round sound when CS2 is not focused.

If you want a sound when a round ends in-game, enable that option.

If you want a rest reminder after a full match ends, enable the rest reminder option.

If the rest notification does not appear, click `Open notifications` and allow notifications.

Safety:

It only uses Valve's official Game State Integration.

It does not read game memory, inject code, simulate mouse or keyboard input, or control gameplay.

It is portable and does not need installation.

The project is open source. The GitHub link is in the description.
